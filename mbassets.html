Asset Details

Name: Palisade Barricade

Asset Description:
The asset comes with one defensive structure in the Palisade Barricade. It is fully textured in a low poly style. It is optimized to operate well in any game. You will also find a low poly sand and low poly water tile in the set to help dress it up with something besides a generic terrain set. Use them to create a large map. The barricade object itself can be rotated in place to create more structures. It is also set up to be placed lower in the ground, and enables the user to alter the height if they desire.

Asset Contents:
- One Textured Low Poly Palisade Barricade
- One Low Poly Sand Tile
- One Low Poly Water Tile
